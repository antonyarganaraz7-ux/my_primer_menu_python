# my_primer_menu_python
# ️ Proyecto: Asistente de Misión Intergaláctica
## 📝 Descripción
Este es mi primer programa escrito en **Python**. Consiste enun menú interactivo que permite gestionar las operaciones
básicas de una nave espacial (o la temática que elijas). El
objetivo de este proyecto fue aprender sobre bucles,
condicionales y cómo organizar código en funciones.

---
## ️ Tecnologías Usadas
* **Lenguaje:** Python 3
* **Editor:** Visual Studio Code / Google Colab
* **Control de Versiones:** Git & GitHub

---
## 🎮 Funcionalidades del Menú
A continuación, se describen las opciones disponibles en el
asistente:
| Opción | Acción | Descripción |
| :--- | :--- | :--- |
| 1 | Verificar Estado | Muestra los niveles de recursos de
la nave. |
| 2 | Navegación | Calcula la distancia al siguiente planeta.
|
| 3 | Configuración | Permite cambiar el nombre de la flota.
|
| 4 | **[Personalizada]** | [Aquí describe la función que TÚ
agregaste]. |
| 5 | Salir | Cierra el programa de forma segura. |

---
## 🚀 Cómo ejecutarlo
Si quieres probar mi programa en tu computadora, sigue estos
pasos:
1. Asegúrate de tener instalado
[Python](https://www.python.org/).
2. Descarga el archivo `asistente.py` de este repositorio.
3. Abre una terminal en la carpeta del archivo.
4. Escribe el comando:
```bash
python asistente.py
```
